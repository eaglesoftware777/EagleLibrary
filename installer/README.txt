WHAT IS EAGLE LIBRARY?
-----------------------
Eagle Library is a professional eBook library manager for Windows.
It lets you manage thousands of books stored in any folder structure
without moving or reorganising your files.

KEY FEATURES
------------
* Fast multi-threaded folder scanner (PDF, EPUB, MOBI, AZW, DjVu, CBZ, and more)
* Auto-fetch metadata (title, author, ISBN, description, rating) from Google Books
* Auto-download cover art
* Beautiful gallery grid view and compact list view
* Advanced search and filtering (format, favourites, text)
* Personal notes, tags, star ratings per book
* SQLite database — portable, no server required
* Fully native Windows application (no .NET required)
* Works on Windows 7, 10 and 11 (32-bit and 64-bit)

GETTING STARTED
---------------
1. Launch Eagle Library from the Desktop or Start Menu shortcut.
2. Go to Settings (⚙ toolbar button or File → Settings).
3. Add the folders where your eBooks are stored.
4. Eagle Library will scan them automatically and show your library.
5. Double-click any book to open the detail panel and edit metadata.

SYSTEM REQUIREMENTS
-------------------
* Windows 7 SP1 / 10 / 11 (64-bit recommended)
* 100 MB disk space (plus space for cover art cache)
* Internet connection (optional — for metadata and cover downloads)
