Database Editor and Troubleshooting

Database Editor

The editor lets you search, scroll, resize columns, move columns, edit metadata fields, reload rows, and delete selected records.

Database Maintenance

Use the Database menu for maintenance actions:

  • Diagnose Text Problems scans the active library for mojibake, replacement glyphs, empty titles, and suspicious metadata.
  • Repair Text Problems rescans only suspicious records and verifies PDF ISBN values from extracted text before saving updates.
  • Optimize Database runs SQLite optimize and analyze to improve browsing speed after large scans or imports.
  • Vacuum Database rebuilds the file and can reduce wasted space after many edits.
  • Reindex Database rebuilds SQLite indexes.

Why Some Titles May Show as Squares

The most common cause is missing glyph coverage in the chosen UI font, not broken SQLite storage. Eagle Library now avoids forcing narrow font families so Qt can use normal fallback fonts for mixed-language titles.

What To Check

  • Whether the same title looks correct in another app on the same machine.
  • Whether the problem affects only one script or language.
  • Whether the stored metadata itself contains replacement characters or OCR garbage.

If the text is truly wrong in the database, run the diagnosis and repair tools first, then use the editor for the remaining exceptions.
