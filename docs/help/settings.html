Settings Reference

This page documents every visible tab and option in the Settings dialog.

Library Folders Tab

  • Library profile selects the library profile being edited.
  • New Library... creates a new top-level library profile.
  • Rename renames the current library profile.
  • Remove deletes the current library profile entry. This removes the profile configuration, not the real files.
  • Folder list shows the watched folders assigned to the selected library.
  • Add Folder... adds another watched folder to the selected library.
  • Remove deletes the selected folder rows from the library profile.

Options Tab

Library Behavior

  • Automatically fetch metadata from internet queues online metadata lookup automatically.
  • Automatically download book covers downloads covers when metadata or cover URLs are found.
  • After scan, enrich only newly added books that still miss DB info limits post-scan enrichment to new weak records.
  • Fast scan mode: index files first and defer deep metadata probing speeds up initial indexing by skipping heavy probing during scan.
  • Grid card width changes the card size in grid view.
  • Scan worker threads controls the number of parallel workers for scan. Auto is usually best.

Interface

  • Startup view chooses whether Eagle remembers the last view or always starts in grid or list mode.
  • Use denser cards and rows to fit smaller screens enables compact mode.
  • Show the browse sidebar on wide screens keeps the left navigation dock visible on larger windows.
  • Show smart category shortcuts in the sidebar displays dynamic category shortcuts.
  • Remember window size and position stores the main window geometry between launches.
  • Language switches the application language in real time. English remains the default.
  • Open Language Packs Folder opens the writable folder where custom JSON language packs can be added.

Language Packs

  • Built-in languages are bundled in the executable and are available immediately.
  • Additional language packs can be added in the translations folder beside the app or in the settings folder.
  • If a string is missing in a custom pack, Eagle Library falls back to the English text automatically.

About Tab

  • Shows the Eagle Library product name and version.
  • Shows the Eagle Software company line.
  • Provides a website link.

Saving and Canceling

  • OK saves all changed settings and closes the dialog.
  • Cancel closes the dialog without saving the current session changes. If you previewed another language, the dialog restores the previous language when canceled.
