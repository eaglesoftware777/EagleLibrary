Libraries and Shelves

A library is a profile with its own watched folders. A shelf is a virtual view inside the active library.

Libraries

  • Create, rename, or remove libraries in Settings.
  • Each library owns its own folder list.
  • The toolbar library selector switches the active workspace.
  • Large libraries scan best with Fast scan mode enabled, then metadata tools can enrich only the records that need deeper analysis.

Shelves

  • All Items
  • Books Only
  • Documents Only
  • Favourites
  • Recently Added
  • Most Opened
  • Missing Metadata
  • No Cover

Shelves do not move files. They only filter or sort the current library view.
