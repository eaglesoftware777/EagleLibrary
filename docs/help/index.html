Quick Start

  1. Open Settings and create a library profile.
  2. Add the folders that belong to that library.
  3. Keep Fast scan mode enabled in Settings for large libraries, then run Scan to index files quickly.
  4. Use the Shelf selector for views like Books Only, Documents Only, and Missing Metadata.
  5. Use Fetch All Metadata, Repair Text Problems, Smart Categorize, and Smart Rename to improve the catalog after the fast scan.
  6. Use Settings -> Language to switch the interface in real time. English is the default language.
  7. Export the active library to JSON, CSV, BibTeX, or RIS.

New In This Build

  • Fast scan mode indexes files first and defers expensive PDF probing until later tools need it.
  • The format sidebar now includes Microsoft Office document shortcuts such as Word, Excel, PowerPoint, Visio, Publisher, Project, Access, and OneNote.
  • Database -> Optimize Database runs SQLite optimize and analyze for faster browsing after large imports.
  • Help -> Install PDF/OCR Tools... opens trusted setup paths for Poppler pdftotext tools and Tesseract OCR.
  • Reference URLs are now built only from trusted whitelisted domains.
  • The application now includes built-in English, French, Spanish, Arabic, and Simplified Chinese language support.
  • Custom language packs can be added as JSON files in the translations folder without rebuilding the app.

Supported Formats

Eagle Library indexes common ebook and document archives, including PDF, EPUB, MOBI, AZW/AZW3, DjVu, FB2, CBZ/CBR, TXT, RTF, Markdown, CSV, CHM, LIT, HTML, XML, Apple iWork, OpenDocument, OneNote, XPS/OpenXPS, and Microsoft Office files.

Microsoft Office support includes Word, Excel, PowerPoint, Visio, Publisher, Project, Access, and Outlook message files. These can be filtered from the toolbar or the browse sidebar after scanning.
