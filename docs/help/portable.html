Portable Build and Packaging

The portable build keeps data and settings beside the executable when portable.flag or portable.ini is present.

Portable Layout

  • data/ for the SQLite database, covers, and JSON reports
  • settings/ for the INI file
  • help/ for the CHM or HTML help
  • translations/ for optional custom JSON language packs

CHM Build

Compile EagleLibrary.hhp with Microsoft HTML Help Workshop to generate EagleLibrary.chm.

Bundled Tools

The portable package does not automatically ship Poppler or Tesseract. Use Help -> Install PDF/OCR Tools... on the target machine to open trusted install pages or launch the OCR installer command.
