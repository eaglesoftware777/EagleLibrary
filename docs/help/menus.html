Menus and Toolbar

This page documents the main menu bar, toolbar, and the most important shortcuts in Eagle Library.

File Menu

  • Settings... opens the full settings dialog.
  • Export Library... exports the active library to JSON, CSV, BibTeX, or RIS.
  • Import Library... imports a previously exported library snapshot.
  • Stop All Tasks stops scan, metadata, cover, and rename tasks. Shortcut: Ctrl+.
  • Quit closes the application.

Library Menu

  • Scan Folders indexes the folders of the active library. Shortcut: F5.
  • Fetch All Metadata fetches metadata for the active library.
  • Advanced Search... opens the advanced search dialog. Shortcut: Ctrl+Shift+F.
  • Reference Views -> Current Selection... opens trusted external reference searches for the selected item.
  • Enrich Incomplete Books refreshes weak or missing metadata only.
  • Quality Check produces a quality report for the active library.
  • Find Duplicates looks for duplicate files by hash, ISBN, and metadata similarity.
  • Extract ISBNs + Fetch Metadata tries to recover ISBN values and queue metadata fetches.
  • Count Pages updates page counts where possible.
  • Generate Missing Covers creates missing cover art placeholders or extracted covers.
  • Normalize Covers standardizes saved cover assets.
  • Smart Categorize classifies items as books or documents and suggests categories.
  • Smart Rename All... repairs weak titles across the active library. Shortcut: Ctrl+R.
  • Smart Rename Selected... repairs only the current selection. Shortcut: Ctrl+Shift+R.
  • Remove Missing Files deletes records whose files no longer exist on disk.
  • Remove Selected removes the selected records from the library. Shortcut: Delete.

View Menu

  • Grid View shows cards with cover art. Shortcut: Ctrl+G.
  • List View shows denser rows. Shortcut: Ctrl+L.
  • Refresh reloads the active library from the database.

Database Menu

  • Database Summary shows basic library and database statistics.
  • Diagnose Text Problems finds mojibake, replacement glyphs, suspicious titles, and similar text issues.
  • Repair Text Problems... rescans suspicious records and updates recoverable metadata.
  • Open Database Folder opens the folder that contains the SQLite database.
  • Database Editor... opens the built-in editor for manual row edits.
  • Optimize Database runs SQLite optimize and analyze for better performance.
  • Vacuum Database rebuilds the SQLite file to reclaim space.
  • Reindex Database rebuilds indexes after heavy changes.

Help Menu

  • About Eagle Library shows version and company details.
  • User Guide opens the CHM or HTML documentation set.
  • Install PDF/OCR Tools... checks for pdftotext, pdftoppm, and tesseract and opens trusted install paths.
  • Visit Website opens the Eagle Software website.

Toolbar

  • Library switches the active library profile.
  • Shelf switches virtual views such as All Items, Books Only, and Missing Metadata.
  • Format narrows the current view by file format.
  • Category narrows the current view by category or classification.
  • Sort changes the database sort field.
  • Asc flips the sort direction.
  • Grid and List toggle the main view mode.
  • Favourites toggles the favourites filter.
  • Settings opens settings directly from the toolbar.
  • Advanced Search opens the structured search dialog.
  • The search box supports free text plus advanced tokens such as author:, title:, tag:, category:, isbn:, and path:.

Live Language Switching

  • The interface ships with English, French, Spanish, Arabic, and Simplified Chinese.
  • The menu bar, toolbar, sidebar, splash messages, workspace header, and settings dialog update immediately after the language changes.
  • Arabic also switches the application layout to right-to-left while active.
