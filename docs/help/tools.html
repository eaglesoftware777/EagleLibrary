Scanning, Metadata, and Smart Rename

Fast Scan Mode

Fast scan mode is designed for large libraries. It stores the file path, format, size, hash, and filename title first, then leaves heavier PDF probing for later tools such as metadata fetch, repair, and smart rename. This makes first-time indexing much faster.

Scan Worker Threads

Use Auto for most systems. Use lower presets when the PC becomes slow during scanning. Use higher presets only when storage and CPU are fast.

Metadata

Eagle Library can fetch metadata, covers, ISBNs, page counts, and tags. The metadata tools can work on the whole active library or on the current selection.

Smart Rename

Smart Rename now prefers content-derived titles before weak filenames. For PDFs it tries embedded metadata, text extraction, pdftotext, and OCR fallbacks when available.

Installing PDF and OCR Tools

Open Help -> Install PDF/OCR Tools... to check whether pdftotext, pdftoppm, and tesseract are available. The dialog opens the trusted Poppler download pages for PDF text extraction tools and can launch a Windows winget install command for Tesseract OCR.

Reference Views

Use the Library menu or the item context menu to open related references for the selected item on Wikipedia, Amazon Books, Open Library, Google Books, WorldCat, and the Library of Congress. Eagle now shows only syntactically valid URLs from trusted whitelisted domains.
